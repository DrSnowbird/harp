# Fascia-SubgraphCounting
Research effort to improve the Fascia Subgraph-Counting algorithm

## How to Run
1. Git clone this repository
1. Browse into the fascia-1.0 directory
1. Check the Makefile (changing it if needed)
1. Execute the following in the terminal - `make clean`
1. Execute the following in the terminal - `make`
1. This will create an executable file called fascia
1. General case: `./fascia -g <graph location i.e. /home/miami.graph> -t <template location i.e. /home/template.fascia>`
1. Check out http://fascia-psu.sourceforge.net/ for more usage examples
1. Check out run-me folder for more detailed information on how to run on Juliet/Tango
